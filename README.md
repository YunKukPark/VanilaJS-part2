## 1.0 Project Setup

> 기본 HTML 과 CSS 셋업

### 1. HTML

1. Canvas 생성
2. Controler 생성 (range, buttons, color-picker 모음)
3. Color-picker 모듈 생성
4. Fill, Save 버튼 모듈 생성
5. input -> range 모듈 생성

### 2. CSS

1. **body** 는 `100vh` 로 관리
2. **왠만한 정렬**은 모두 `flex` 로 처리
3. **button** 과 **color-picker** 들은 `transition` 을 주어 스르륵 `hover` 되었다는걸 알림.
